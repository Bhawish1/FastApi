## Overview
Backend-focused Todo application built with FastAPI.

## Backend
- Designed and implemented REST APIs using FastAPI
- JWT-based authentication and authorization
- Secure password hashing (bcrypt)
- PostgreSQL database integration
- Environment-based configuration for secrets
- Deployed backend with OpenAPI documentation

## Frontend
- It has been sourced
- Minimal frontend UI sourced/adapted only to demonstrate backend functionality
- Backend is API-first and frontend-agnostic
